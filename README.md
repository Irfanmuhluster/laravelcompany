<h1><b>Langkah-langkah Install</b></h1>

- Composer Install
- npm install && npm run dev
- Daftarkan nama db, username, password di .env
- php artisan key:generate
- php artisan migrate
- php artisan db:seed
- php artisan serve
- kunjungi http://127.0.0.1:8000/admin/
