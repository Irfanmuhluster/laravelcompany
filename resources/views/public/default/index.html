Fitur CMS Wordpress
CMS wordpress adalah cms yang mudah digunakan cocok untuk landing page dan website company profile yang sederhana.
  • Customisasi Landing Page / Halaman depan
  • Artikel / Post
  • Halaman / Page
Fitur CMS Toko Online WA
Anda lebih sering menggunakan WA untuk berinteraksi dengan pelanggan?

Mungkin fitur ini cocok untuk website penjualan Anda. Hanya menampilkan kataog produk, saat pelanggan menekan beli maka langsung membuka aplikasi WA dan link detail produk akan tercantum disitu. Anda akan berinteraksi dengan pelanggan melalui WA.
  • Customisasi Landing Page / Halaman depan
  • Artikel / Post
  • Halaman / Page
  • Katalog Produk
  • Kategori Produk
  • Detail Produk
  • Hubungi Kami Via WA
Fitur CMS Wordpress + Woocomerce

  • Halaman / Page
